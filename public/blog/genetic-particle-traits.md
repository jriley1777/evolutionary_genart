# Genetic Particle Traits

Evolutionary particle system with visual DNA, genetic mutations, and behavioral diversity.

## Concept

- Each particle has a “DNA” that defines shape, color, size, speed, and behavior.
- Offspring inherit traits from parents with a chance of mutation.
- Behaviors can include flow-following, swarming, orbiting, pulsing, or spiraling.
- Special abilities (e.g. attract, repel, multiply) add more interaction.

## Technical notes

- DNA is a small set of parameters; inheritance copies parent values and may mutate.
- Behavior is a mode that changes how forces are applied each frame.
- Clicking can spawn new particles or inject variety to keep the population evolving.
